# 🎮 گیم‌نت — سیستم مدیریت مرکز بازی

سیستم مدیریت حرفه‌ای برای بیلیارد، اسنوکر، PS4/PS5 و کافه.

---

## راه‌اندازی سریع

```bash
pip install -r requirements.txt
python manage.py migrate        # فقط بار اول
bash run.sh
```

مرورگر را باز کنید: **http://127.0.0.1:8000**

| فیلد | مقدار |
|------|-------|
| نام کاربری | `admin` |
| رمز عبور | `admin123` |

---

## تغییر رمز عبور

```bash
python manage.py changepassword admin
```

---

## بکاپ

کافی است فایل `db.sqlite3` را کپی کنید — تمام داده‌های سیستم در آن است.

## امکانات

### داشبورد
- نمایش همه دستگاه‌ها: آزاد (سبز) / در حال استفاده (قرمز) / پرداخت‌نشده (زرد)
- تایمر زنده برای سشن‌های فعال
- هشدار سشن‌های بدون پرداخت
- هشدار موجودی کم

### سشن‌ها
- شروع سشن با انتخاب بازیکن (ثبت‌نام‌شده یا مهمان)
- پایان سشن و محاسبه خودکار مبلغ
- **پرداخت انعطاف‌پذیر:**
  - افزودن چند پرداخت جداگانه با مبلغ دلخواه
  - ترکیب نقد + حساب در یک سشن
  - پرداخت جزئی و بازگشت بعد برای تکمیل
  - دکمه «بعداً پرداخت کنم» — سشن در داشبورد زرد می‌شود
  - حذف پرداخت اشتباه

### مشتریان
- ثبت حساب مشتری با نام و تلفن
- سیستم بدهی/اعتبار
- تاریخچه سشن، خرید و پرداخت

### بدهی‌ها
- لیست کامل بدهکاران
- تسویه سریع از همان صفحه

### فروش کافه
- کلیک روی محصول → انتخاب تعداد → نقد یا حساب
- لینک اختیاری به سشن فعال

### گزارش‌ها
- امروز / ۷ روز / ۳۰ روز
- درآمد نقد vs حساب
- پرفروش‌ترین محصولات
- فعال‌ترین دستگاه‌ها و مشتریان
